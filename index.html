Q1. function counter(){

    function IncreaseCounter() {
        return counter += 1;
    };

    return IncreaseCounter;
}

var counter = counter();
alert(counter());
alert(counter());
alert(counter());
alert(counter());
OutPut: 1 2 3
We will get output in alert box one after another as mentioned above after the acceptance of one output(1,2,3,4). We have to click on "OK" button to get the next output(value).

Q5. Take a variable in outer function and create an inner function to increase the counter every time it is called

ans: function otFun(l){
            function inFun(){
                l++;
                console.log(l);
            }
            inFun();
            inFun();
            inFun();
        }
        otFun(50);

        Output : 51 52 53
